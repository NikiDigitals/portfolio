# NikiDigitals Portfolio

This repository contains the technical artefacts that support the NikiDigitals professional portfolio.

## Purpose

The repository serves as the technical layer of the portfolio and contains:

* Source code
* Jupyter notebooks
* Technical reports
* Data analysis projects
* Financial models
* Demonstrations and prototypes
* Project deliverables
* Supporting technical documentation

## Portfolio Architecture

NikiDigitals uses a two-layer portfolio architecture.

### GitHub — Technical Layer

GitHub contains the implementation and evidence layer of projects, including code, notebooks, technical documentation, models, reports, and project outputs.

### GitBook — Thinking Layer

The NikiDigitals Portfolio GitBook contains the reasoning behind projects, including:

* Problem definition
* Research process
* Methodology
* Decision-making
* Systems thinking
* Reflections and lessons learned
* Strategic and organisational implications

Together, these platforms provide both the implementation and the thinking behind each project.
